structure

Cathedral rules

Foundations — Logic/Set/Discrete + TOC
The spine. Axioms-first thinking in logic and set theory, formal proof methods, and the full Sipser arc from automata through computability to complexity. This is what makes the rest rigorous.
  • Propositional and predicate logic — axioms, proof templates, ∀/∃
  • Set theory — sets, relations, functions, cardinality
  • TOC — automata, decidability, NP, reductions
  • Mathematical logic — completeness theorem (Enderton)
Math — Linear Algebra + Probability
Axler's Linear Algebra Done Right: vector spaces, linear maps, eigenvalues, inner products. Selected probability. Both feed quantum computing and theoretical ML foundations.
  • Vector spaces, linear maps, kernel/image
  • Spectral theorem, inner products, unitary operators
  • Probability axioms, Bayes, expectation, Markov chains
Algorithms/DSA — kept sharp
Patterns and correctness arguments, not just code. Invariants, induction, reduction schemas. Algorithms as theorems.
  • Hashmaps, two pointers, sliding window
  • BFS/DFS, heaps, greedy proofs
  • DP basics, invariants and correctness
Quantum Thread — small, consistent, never dies
Nielsen & Chuang as reference. Weekly thread — tiny, consistent. The goal is to arrive at de Wolf's course ready.
  • Postulates, gates, circuits
  • Entanglement + teleportation
  • Deutsch–Jozsa, Grover, Shor (conceptual)
  • BQP intuition
